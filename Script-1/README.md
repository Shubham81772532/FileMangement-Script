# Directory Management Script (File-Utility-Script.sh.sh)

## Usage
File-Utility-Script.sh <action> <path>[dirname]

## Actions
addDir     -> Create directory  
deleteDir  -> Delete directory  
listDir    -> List directories   
listFiles  -> List files  
listAll    -> List files + directories  

## Commands
File-Utility-Script.sh addDir /path dirname  
File-Utility-Script.sh deleteDir /path dirname  
File-Utility-Script.sh listDir /path  
File-Utility-Script.sh listFiles /path  
File-Utility-Script.sh listAll /path  

## Requirements
chmod +x file2.sh  

## Notes
Uses: mkdir, rm, ls, find, realpath  
rm -rf is permanent (use carefully)
